### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sifatullahsu/education-server-rainier_technologies.git
   cd education-server-rainier_technologies
   ```

2. Install project dependencies:

   ```bash
   yarn install
   ```

3. Run the project

   ```bash
   yarn dev
   ```
